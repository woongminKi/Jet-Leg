→ 회수 비용 ~1.5h → **30분으로 단축** (단순 aggregate 추가).

### 1.2 가중 방식

각 버킷 sample_count 로 가중 평균:
```
overall_pass_rate = Σ(bucket.pass_rate × bucket.sample_count) / Σ bucket.sample_count
```

**근거**: 단순 평균이면 sample 1건 버킷이 sample 100건 버킷과 동일 가중 → 부정확. sample_count 가중이 §13.1 "전체 인제스트 SLO 달성률" 의미 정확.

## 2. 구현

### 2.1 변경 파일 (`stats.py`)

```python
# 신규 모델
class IngestSloAggregate(BaseModel):
    total_samples: int
    overall_pass_rate: float | None
    buckets_with_samples: list[str]

# StatsResponse 확장
ingest_slo_aggregate: IngestSloAggregate

# 신규 helper
def _compute_slo_aggregate(slo_buckets):
    weighted_sum = 0.0
    total = 0
    bs = []
    for name, bucket in slo_buckets.items():
        if bucket.sample_count > 0 and bucket.pass_rate is not None:
            weighted_sum += bucket.pass_rate * bucket.sample_count
            total += bucket.sample_count
            bs.append(name)
    return IngestSloAggregate(
        total_samples=total,
        overall_pass_rate=round(weighted_sum / total, 4) if total > 0 else None,
        buckets_with_samples=bs,
    )
```
