### 5.1 P0/P1 — 검색 성능 향상 plan (2026-05-05 senior-planner + senior-qa 교차검증)

**현재 측정**: 검색 적합도 (BGE-M3 cosine + DCG 가중) **0.60 (60점)**.
**핵심 진단 (qa)**: BGE-M3 한국어 cosine 분포에서 0.6 = "주제 동일·직접 답변 직전" **정상 분포** (강관련 0.65~0.80). 사용자가 "낮다" 느끼는 진짜 원인:
1. Scale mismatch — "100점 만점" metaphor 오해
2. 비교형 query 한계 — "각 삼국 차이" 같은 비교 의도는 cosine 매칭 X
3. DCG 가중 효과 — top-1=0.7 + top-10=0.5 가중평균 = 0.60

| 순서 | 후보 | 근거 | 작업량 | 기대 |
|---|---|---|---|---|
| **S0 즉시** | **UX 라벨 변경** (qa) — "검색 적합도 60%" → "의미 유사도 0.60 (참고치)" + Top-3 평균 + Max 병기 | 알고리즘 변경 0, 사용자 오해 즉시 해소 | 下 (~30분) | UX ↑ |
