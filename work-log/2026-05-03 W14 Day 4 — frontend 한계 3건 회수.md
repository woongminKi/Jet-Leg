### 2.2 NoResults 분기 (한계 #68)

```tsx
const isDocScope = docId !== null;
return (
  <div>
    <p>{isDocScope ? `이 문서 안에서 '${query}' 결과가 없어요` : ...}</p>
    <p>{isDocScope ? '문서 스코프를 해제하고 전체에서 다시 찾아보세요.' : ...}</p>
    {isDocScope && <Link href={`/search?q=${...}`}>전체 검색으로 보기</Link>}
    {!isDocScope && popularTags.length > 0 && <PopularTagsBadges />}
  </div>
);
```

### 2.3 by_mode mini-row (한계 #83)

```tsx
{slo.by_mode && Object.values(slo.by_mode).some(m => m.sample_count > 0) && (
  <ul>
    {['hybrid', 'dense', 'sparse'].map(m => {
      const entry = slo.by_mode?.[m];
      if (!entry || entry.sample_count === 0) return null;
      return <li>{m}: p50 {entry.p50_ms}ms · n {entry.sample_count}</li>;
    })}
  </ul>
)}
```

→ 사용자가 ablation 후 SearchSloCard 에서 mode 별 p50 직접 비교 가능.

### 2.4 mode 토글 반응형 (한계 #78)

```tsx
// 이전: hidden md:inline-flex (mobile 미노출)
// 이후: inline-flex h-6 md:h-7 — mobile 폰트/패딩 축소
className={mode === m
  ? 'h-full bg-primary px-1.5 text-[9px] md:px-2 md:text-[10px]'
  : 'h-full px-1.5 text-[9px] md:px-2 md:text-[10px]'}
```
